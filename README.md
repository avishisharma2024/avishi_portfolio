## Sections Included

### Home
Contains introduction and professional summary with typing animation.

### Education
Displays academic qualifications including university and school details.

### Skills
Lists programming languages, frameworks, and technical competencies.

### Projects
Shows project descriptions and technologies used.

### Experience
Includes internship experience and responsibilities.

### Achievements
Highlights competitions and recognitions.

### Contact
Provides phone number, email, GitHub, and LinkedIn links.

---

## Development Steps

### Step 1: Planning
The layout and sections of the portfolio were planned before development.

### Step 2: Creating HTML Structure
Semantic HTML tags were used to structure the webpage including header, navigation, sections, and footer.

### Step 3: Styling with CSS
CSS was used to apply layout, spacing, borders, and typography. Flexbox was used for alignment.

### Step 4: Adding Interactivity
JavaScript was added to create typing animation in the home section.

### Step 5: Testing
The webpage was tested for layout alignment, navigation links, and responsiveness.

### Step 6: Uploading to GitHub
All project files were uploaded to GitHub repository.

### Step 7: Deployment
The website was deployed using GitHub Pages.

## Challenges Faced

1. Incorrect placement of HTML tags

2. Navigation bar alignment issues  
Solved using CSS Flexbox.

3. Spacing and layout problems  
Resolved using margin, padding, and border properties.

4. Deployment confusion  
Fixed by correctly configuring GitHub Pages.

## Importance of Personal Webpage

A personal webpage helps in building a strong online presence. It acts as a digital portfolio that showcases skills, education, and projects. It allows recruiters and collaborators to easily evaluate technical abilities. A portfolio website also helps in professional branding and makes it easy to share information using a single link. It demonstrates web development knowledge and provides a platform that can be updated regularly as skills and experience grow.
